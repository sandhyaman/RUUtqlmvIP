# Python在线可视化学习系统简介

## 核心功能
- 登录注册
- 学习资源数据可视化
- 学习视频
- 学习资源
- 在线测试
- 通知公告
- 用户管理
- 学习视频管理
- 图书资源管理
- 学习情况管理
- 试题管理

## 适用场景
- 教育机构
- 在线学习平台
- 个人学习者

## 技术特点
- 采用Django框架进行后端开发
- 使用MySQL数据库进行数据存储
- 前端采用Vue.js进行界面展示
- 提供源码、SQL、数据库文档和开发文档，方便用户理解和二次开发
